# cli-hub-meta-skill — A marketplace of agent-native CLIs

> Discover and install ready-made agent-native CLIs for professional software across creative, productivity, and AI workflows. One command to search, one to install.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
CLI Hub is a marketplace meta-skill that connects agents to a growing catalog of agent-native CLIs for professional software. Instead of building a harness from scratch, search and install pre-made CLIs in a single command. The catalog is served live so available tools stay current. Categories span creative, productivity, AI, communication, and development workflows.

## Features
- **Live catalog** — always up-to-date, remote SKILL.md
- **Commands** — `cli-hub list`, `cli-hub search <keyword>`, `cli-hub install <name>`, `cli-hub info <name>`
- **Categories** — creative (image editing, 3D modeling, video, audio, music notation), productivity (office, knowledge management, live streaming), AI (local LLMs, image gen, AI APIs), communication, development

## Usage / Quick Start
```bash
pip install cli-anything-hub

cli-hub list                    # browse all available CLIs
cli-hub search image            # search by keyword
cli-hub install gimp            # install a CLI
cli-hub info gimp               # show details
```
Live catalog: `https://reeceyang.sgp1.cdn.digitaloceanspaces.com/SKILL.md`
